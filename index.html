Módulo 1: Primeiros Passos e o "Modo Pensar"

Seja bem-vindo ao mundo da programação! Vamos aprender como o Python lida com informações básicas usando variáveis clínica veterinária como exemplo:

nome_paciente = "Bruce Brian"  # Texto (String)
idade = 5                      # Número Inteiro (int)
print("Paciente:", nome_paciente, "| Idade:", idade)
  • Variáveis: Servem para guardar dados. O = é o comando de guardar.
  • print(): Mostra o que está guardado no monitor.

Módulo 2: Tomando Decisões e Repetições

O código pode decidir caminhos e repetir tarefas automaticamente:

temperatura = 39.5
if temperatura > 39.2:
    print("Alerta: O animal está com febre!")

sintomas = ["tosse", "apatia"]
for s in sintomas:
    print("Sintoma detectado:", s)
  • if: Executa uma ação apenas "SE" a condição for verdadeira.
  • for: Repete o comando para cada item de uma lista [].

Módulo 3: Organizando com Funções

Funções são "receitas" que você cria uma vez e usa sempre que precisar:

def calcular_dose(peso):
    return peso * 0.5

resultado = calcular_dose(10) # Saída: 5.0
  • def: Define uma nova função/ação personalizada.
  • return: Arremessa o resultado final de volta para o programa.
