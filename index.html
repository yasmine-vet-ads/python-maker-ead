Módulo 1: Primeiros Passos e o "Modo Pensar"

Seja muito bem-vindo! Pense no código como uma sequência de instruções que ensina o computador a resolver problemas passo a passo. O computador apenas lê linha por linha e executa exatamente o que você mandou.

Por que Python?

  • Sintaxe Limpa: O código parece inglês fluente. É feito para humanos lerem sem esforço.
  • Superpoderes de Mercado: Python é uma das linguagens mais utilizadas no mundo em áreas como Inteligência Artificial, automação, análise de dados e Internet das Coisas (IoT).

Conceitos Fundamentais

O comando básico para mostrar qualquer informação na tela do computador é a função print():

print("Olá, Mundo!")

Variáveis são espaços reservados na memória usados para armazenar informações temporariamente. O Python identifica o tipo de dado sozinho:

  • String (str): Texto delimitado por aspas. Ex: nome = "Rex"
  • Inteiro (int): Números inteiros. Ex: idade = 4

Módulo 2: Tomando Decisões e Repetições

As estruturas condicionais permitem que o programa tome decisões com base em condições. if, elif, e else. Lembre-se que o Python usa 4 espaços de recuo (indentação). A indentação (recuo do código) é obrigatória para organizar os blocos de execução.

temperatura = 39.5
if temperatura > 39.2:
    print("O animal está com febre.")
else:
    print("Temperatura normal.")

Estruturas de Repetição (Loops) e Listas

O loop for repete um trecho de código um número definido de vezes. O loop while repete enquanto uma condição for verdadeira.

Listas servem para guardar vários dados juntos usando colchetes []. Exemplo: sintomas = ["tosse", "febre"]. Para adicionar itens, usamos o comando append().

Módulo 3: Organizando a Casa com Funções

Funções são blocos autônomos que realizam tarefas específicas. Criamos uma função utilizando a palavra-chave def e retornamos a resposta com return.

def calcular_imc(peso, altura):
    imc = peso / (altura * altura)
    return imc

O Ecossistema de Bibliotecas

O Python possui ferramentas nativas prontas para uso. Ao usar o comando import, você traz inteligência extra para o seu código. Por exemplo, import time permite criar pausas na execução do código.

Módulo 4: Projeto Integrador Maker (Semáforo)

Agora vamos sair da tela de texto e controlar o mundo real usando MicroPython no simulador de placas Wokwi.

🛠️ O que é a Cultura Maker?

A Cultura Maker (o famoso "faça-você-mesmo") é um movimento global que encoraja as pessoas a criarem, consertarem e modificarem objetos e sistemas físicos usando tecnologia acessível. A ideia principal é que deixemos de ser apenas consumidores passivos de tecnologia para nos tornarmos criadores ativos!

Entendendo a Lógica do Código

  • Configuração (Setup): Usamos Pin(NUMERO, Pin.OUT) para definir que o pino vai enviar energia e Pin(NUMERO, Pin.IN) para o botão, que vai escutar a entrada.
  • O Loop Infinito (while True:): Um semáforo nunca dorme! Este comando cria um laço contínuo.
  • A Detecção (if botao.value() == 0:): É o gatilho da travessia. O código fica "vigiando" o botão.
  • O Efeito Pisca-Pisca (for contador in range(1, 14):): Automatiza a repetição do piscar do sinal do pedestre.

🔌 Passo a Passo: Como Iniciar a Simulação no Wokwi

  1. Acesse wokwi.com, role a página um pouco para baixo e clique em ESP32 na aba MicroPython.
  2. No menu superior do simulador, use o botão azul "+" (Add a new part) para adicionar ao painel: 3 LEDs para os carros, 2 LEDs para os pedestres, resistores e 1 Botão (Pushbutton).
  3. A Magia dos Resistores: Um LED liga direto na placa queima na hora! Você deve usar um resistor em série com o negativo de cada LED para funcionar como um "pedágio" que estrangula a corrente e protege a luz.
  4. Montagem: Conecte a perna positiva de cada LED no seu respectivo pino do ESP32 (ex: Pino 32 para o Vermelho do carro). A perna negativa vai para o Terra (GND) passando pelo resistor.
  5. Cole todo o código-fonte exibido abaixo na aba main.py do simulador e clique no botão verde de Play no Wokwi!
main.py - Código Fonte
from machine import Pin import time # Definindo os pinos sinalVmCarro = Pin(32, Pin.OUT) sinalAmCarro = Pin(25, Pin.OUT) sinalVdCarro = Pin(33, Pin.OUT) sinalVmPedestre = Pin(21, Pin.OUT) sinalVdPedestre = Pin(22, Pin.OUT) botao = Pin(18, Pin.IN) contagem = 13 # Setup Inicial sinalVmCarro.value(0) sinalAmCarro.value(0) sinalVdCarro.value(0) sinalVmPedestre.value(0) sinalVdPedestre.value(0) print("LIBERADO PARA O MOTORISTA") while True: sinalVdCarro.value(1) sinalVmPedestre.value(1) if botao.value() == 0: print("ATENÇÃO MOTORISTA, PREFERENCIA DO PEDESTRE") time.sleep(2) sinalVdCarro.value(0) sinalAmCarro.value(1) time.sleep(2) sinalAmCarro.value(0) sinalVmCarro.value(1) sinalVmPedestre.value(0) sinalVdPedestre.value(1) time.sleep(5) for contador in range(1, 14): sinalVdPedestre.value(1) time.sleep(0.5) sinalVdPedestre.value(0) time.sleep(0.5) contagem -= 1 sinalVmCarro.value(0) contagem = 13 time.sleep(0.1)
