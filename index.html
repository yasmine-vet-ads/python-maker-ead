Módulo 1: Primeiros Passos e o "Modo Pensar"

Seja muito bem-vindo! Pense no código de programação como uma receita de bolo detalhada: o computador apenas lê linha por linha e executa exatamente o que você mandou.

Por que Python?

  • Sintaxe Limpa: O código parece inglês fluente. É feito para humanos lerem sem esforço.
  • Superpoderes de Mercado: Ferramenta nº 1 para Inteligência Artificial (IA) e Internet das Coisas (IoT).

Conceitos Fundamentais

O comando básico para mostrar qualquer informação na tela do computador é a função print():

print("Olá, Mundo!")

Variáveis funcionam como "caixinhas" com etiquetas na memória para guardar dados temporários. O Python identifica o tipo de dado sozinho:

  • String (str): Texto delimitado por aspas. Ex: nome = "Bruce Brian"
  • Inteiro (int): Números inteiros sem casas decimais. Ex: idade = 5

Módulo 2: Tomando Decisões e Repetições

Para darmos inteligência ao código, precisamos que ele tome decisões usando as estruturas if, elif, e else. Lembre-se que o Python usa 4 espaços de recuo (indentação) para organizar os blocos.

temperatura = 39.5
if temperatura > 39.2:
    print("O animal está com febre.")
else:
    print("Temperatura normal.")

Estruturas de Repetição (Loops) e Listas

O loop for repete um trecho de código um número definido de vezes. O loop while repete enquanto uma condição for verdadeira.

Listas servem para guardar vários dados juntos usando colchetes []. Exemplo: sintomas = ["tosse", "febre"]. Para adicionar itens dinamicamente, usamos o comando append().

Módulo 3: Organizando a Casa com Funções

Funções são blocos autônomos que realizam tarefas específicas. Criamos uma função utilizando a palavra-chave def e retornamos a resposta com return.

def calcular_dosagem(peso_animal):
    dosagem = peso_animal * 0.5
    return dosagem

O Ecossistema de Bibliotecas

O Python possui ferramentas nativas prontas para uso. Ao usar o comando import, você traz inteligência extra para o seu código. Por exemplo, import time permite criar pausas na execução do código com o comando time.sleep(2).

Módulo 4: Projeto Integrador Maker (Semáforo)

Agora vamos sair da tela de texto e controlar o mundo real usando MicroPython no simulador de placas Wokwi.

🛠️ O que é a Cultura Maker?

A Cultura Maker (o "faça-você-mesmo") é um movimento global que encoraja as pessoas a criarem e modificarem sistemas físicos usando tecnologia acessível. Deixamos de ser apenas consumidores para virar criadores!

Entendendo a Lógica do Código

  • Configuração (Setup): Usamos Pin(NUMERO, Pin.OUT) para emitir energia e Pin(NUMERO, Pin.IN) para ler botões.
  • O Loop Infinito (while True): Mantém o semáforo rodando continuamente.
  • A Magia dos Resistores: LEDs ligados direto sem resistor queimam na hora! Eles servem como um pedágio para limitar a corrente elétrica perigosa.
main.py - Código Fonte
from machine import Pin import time sinalVmCarro = Pin(32, Pin.OUT) sinalAmCarro = Pin(25, Pin.OUT) sinalVdCarro = Pin(33, Pin.OUT) botao = Pin(18, Pin.IN) while True: sinalVdCarro.value(1) if botao.value() == 0: time.sleep(2) sinalVdCarro.value(0) sinalAmCarro.value(1) time.sleep(2) sinalAmCarro.value(0) sinalVmCarro.value(1) time.sleep(5) sinalVmCarro.value(0)
