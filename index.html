MÓDULO 1: PRIMEIROS PASSOS

Seja bem-vindo ao mundo da programação! Aprender Python é como aprender um novo idioma para dar ordens ao computador. Vamos começar criando um cadastro clínico simples para um pet:

nome_pet = "Bruce Brian"  # Criamos uma String (texto)
idade = 5                 # Criamos um Inteiro (número)
print("Paciente:", nome_pet, "| Idade:", idade, "anos")
  • Variáveis: O sinal de = guarda o valor na memória.
  • Saída: A função print() exibe os dados para o usuário.

MÓDULO 2: DECISÕES E REPETIÇÕES

O computador pode tomar decisões sozinho usando o if (SE) e o else (SENÃO). Veja como detectar febre em um animal:

temperatura = 39.5
if temperatura > 39.2:
    print("O animal está com febre!")
else:
    print("Saúde normal.")
  • Indentação: O espaço de 4 espaços à direita é obrigatório no Python para indicar que o comando está dentro do "if".
AVALIAÇÃO DE FIXAÇÃO:

MÓDULO 3: FUNÇÕES E BIBLIOTECAS

Funções são blocos que você escreve uma vez e usa em qualquer lugar. Usamos def para criá-las e import para trazer ferramentas prontas.

import time

def calcular_dose(peso):
    return peso * 0.5

print("Calculando...")
time.sleep(2) # Pausa de 2 segundos
print("Dose recomendada:", calcular_dose(10), "mg")
AVALIAÇÃO DE FIXAÇÃO:

Qual palavra-chave é usada para devolver o resultado de uma função para o programa principal?
