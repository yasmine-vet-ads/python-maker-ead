# 🩺🐍 Python Start Health

**Curso introdutório de programação para saúde, dados e automação**

O **Python Start Health** é uma versão temática do curso introdutório de Python, voltada para aplicações em saúde, medicina veterinária, dados clínicos, automação de rotinas e inovação digital.

Esta versão foi criada dentro do mesmo repositório do curso original, mantendo a plataforma principal ativa e acrescentando uma trilha específica para a área da saúde.

---

## 🎯 Objetivo

Ensinar fundamentos básicos de Python por meio de exemplos aplicados a:

- saúde;
- medicina veterinária;
- dados clínicos;
- automação;
- organização de registros;
- geração de relatórios simples;
- HealthTech e VetTech.

## 📚 Módulos

### Módulo 1 — Primeiros passos em Python aplicado à saúde

Variáveis, tipos de dados, `print()` e criação de fichas simples.

### Módulo 2 — Decisões e alertas

Uso de `if`, `elif` e `else` para criar regras simples de triagem e classificação.

### Módulo 3 — Listas e repetições

Uso de listas, `for`, `while` e `range()` para organizar registros.

### Módulo 4 — Funções e automação

Criação de funções para automatizar cálculos e relatórios simples.
