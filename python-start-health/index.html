Módulo 1 — Python como ficha do paciente

Analogia central: uma variável em Python é como um campo da ficha do paciente. Cada campo guarda uma informação: nome, idade, peso, espécie, exame ou observação.

Programar é criar uma sequência de instruções para que o computador resolva uma tarefa. Na área da saúde, isso pode ajudar a organizar dados, calcular indicadores, padronizar textos e reduzir tarefas repetitivas.

1. O primeiro comando: print()

O comando print() mostra uma informação na tela. Pense nele como uma forma de “imprimir” uma mensagem, um alerta ou uma ficha simples. 

print("Olá! Bem-vindo ao Python Start Health.")

2. Variáveis: campos da ficha

Variáveis guardam informações. Na saúde, podemos imaginar cada variável como um campo do prontuário. 

nome_paciente = "Luna"
especie = "canina"
idade = 8
peso = 12.4

print(nome_paciente)
print(idade)
print(peso)

3. Tipos de dados: tipo de informação registrada

  • str: texto, como nome, espécie e exame.
  • int: número inteiro, como idade.
  • float: número decimal, como peso.
  • bool: verdadeiro ou falso, como tem_febre = True.
nome = "Luna"          # str
idade = 8              # int
peso = 12.4            # float
tem_febre = False      # bool

4. f-string: montando uma ficha automaticamente

A f-string permite misturar texto com variáveis. É muito útil para gerar mensagens, fichas, relatórios e textos padronizados. 

nome_paciente = "Luna"
especie = "canina"
idade = 8
peso = 12.4
exame = "ultrassonografia abdominal"

ficha = f"Paciente {nome_paciente}, espécie {especie}, {idade} anos, {peso} kg. Exame: {exame}."
print(ficha)

Módulo 2 — Python como triagem

Analogia central: o if funciona como uma regra de triagem. Se uma condição for verdadeira, uma ação acontece. Caso contrário, o código segue outro caminho.

Em Python, usamos if, elif e else para criar decisões. Isso é útil para transformar critérios em mensagens automáticas.

1. Estrutura básica

temperatura = 39.4

if temperatura >= 39.2:
    print("Alerta: temperatura elevada.")
else:
    print("Temperatura dentro do intervalo esperado.")

2. Comparações

  • == compara igualdade.
  • != verifica diferença.
  • > maior que.
  • < menor que.
  • >= maior ou igual.
  • <= menor ou igual.

3. Operadores lógicos

Podemos combinar condições com and e or. 

idade = 13
creatinina = 2.1

if idade >= 10 and creatinina > 1.8:
    print("Atenção: paciente geriátrico com alteração renal.")
else:
    print("Sem alerta automático pelos critérios definidos.")

4. Validação de dados

Em sistemas de saúde, validar dados evita erros. Por exemplo: peso não pode ser negativo. 

peso = -2

if peso <= 0:
    print("Erro: o peso precisa ser maior que zero.")
else:
    print("Peso registrado com sucesso.")
Atenção: regras automáticas em sistemas de saúde servem para organizar e alertar. Elas não substituem avaliação profissional.

Módulo 3 — Python como fila de atendimento

Analogia central: uma lista em Python é como uma fila ou relação de pacientes. O for percorre essa fila e executa uma ação para cada item.

Listas e repetições ajudam a processar vários registros automaticamente. Em saúde, isso pode ser usado para percorrer pacientes, exames, imagens, laudos ou resultados.

1. Listas

pacientes = ["Luna", "Rex", "Mel", "Thor"]

print(pacientes)
print(pacientes[0])

2. Loop for: percorrendo a fila

pacientes = ["Luna", "Rex", "Mel", "Thor"]

for paciente in pacientes:
    print(f"Gerando ficha para: {paciente}")

3. Lista de dicionários: prontuários estruturados

Um dicionário guarda informações em pares de chave e valor. É parecido com um prontuário estruturado. 

paciente = {
    "nome": "Luna",
    "especie": "canina",
    "idade": 8,
    "peso": 12.4
}

print(paciente["nome"])
print(paciente["peso"])

4. Vários prontuários em uma lista

pacientes = [
    {"nome": "Luna", "peso": 12.4},
    {"nome": "Rex", "peso": 18.7},
    {"nome": "Mel", "peso": 4.8}
]

for paciente in pacientes:
    print(f"{paciente['nome']} pesa {paciente['peso']} kg.")

Módulo 4 — Python como protocolo automatizado

Analogia central: uma função em Python é como um protocolo. Você define uma sequência de passos uma vez e reutiliza sempre que precisar.
Aviso importante: este módulo é um exemplo educacional de programação. O cálculo demonstrado não substitui prescrição, avaliação clínica, conferência de bula, espécie, contraindicações, concentração real do produto, condição do paciente ou orientação profissional.

1. Funções: protocolos reutilizáveis

def gerar_ficha(nome, especie, idade, peso):
    ficha = f"Paciente {nome}, espécie {especie}, {idade} anos, {peso} kg."
    return ficha

resultado = gerar_ficha("Luna", "canina", 8, 12.4)
print(resultado)

2. Projeto do módulo: calculadora educacional de dose por peso

Agora vamos criar uma calculadora didática. O exemplo usado será:

  • Fármaco do exemplo: dipirona gotas.
  • Dose didática: 12 mg/kg.
  • Concentração exemplo: 500 mg/mL.
  • Conversão exemplo: 20 gotas/mL.

Passo 1 — Dose total em mg

Dose total em mg = peso do animal × dose em mg/kg

Exemplo: 10 kg × 12 mg/kg = 120 mg

Passo 2 — Concentração do medicamento

Concentração exemplo: 500 mg/mL

Isso significa que cada 1 mL contém 500 mg.

Passo 3 — Converter mL em gotas

20 gotas/mL

500 mg/mL ÷ 20 gotas/mL = 25 mg por gota

Passo 4 — Calcular número de gotas

Gotas = dose total em mg ÷ mg por gota

120 mg ÷ 25 mg/gota = 4,8 gotas

Resultado aproximado: 5 gotas.

3. Código em Python

def calcular_dipirona_gotas(peso_kg):
    dose_mg_por_kg = 12
    concentracao_mg_por_ml = 500
    gotas_por_ml = 20

    dose_total_mg = peso_kg * dose_mg_por_kg
    volume_ml = dose_total_mg / concentracao_mg_por_ml
    mg_por_gota = concentracao_mg_por_ml / gotas_por_ml
    numero_de_gotas = dose_total_mg / mg_por_gota

    return dose_total_mg, volume_ml, mg_por_gota, numero_de_gotas


peso = 10

dose_total, volume, mg_gota, gotas = calcular_dipirona_gotas(peso)

print(f"Peso do animal: {peso} kg")
print(f"Dose total: {dose_total} mg")
print(f"Volume calculado: {volume:.3f} mL")
print(f"Cada gota possui aproximadamente: {mg_gota} mg")
print(f"Número calculado de gotas: {gotas:.2f}")
print(f"Número aproximado de gotas: {round(gotas)}")
