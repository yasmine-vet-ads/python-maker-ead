Módulo 2 — Decisões e alertas com estruturas condicionais

Em Python, usamos if, elif e else para criar regras simples.

Exemplo básico

temperatura = 39.4

if temperatura >= 39.2:
    print("Alerta: temperatura elevada.")
else:
    print("Temperatura dentro do intervalo esperado.")

Exemplo com duas condições

idade = 13
creatinina = 2.1

if idade >= 10 and creatinina > 1.8:
    print("Atenção: paciente geriátrico com alteração renal.")
else:
    print("Sem alerta automático pelos critérios definidos.")
Estes exemplos são educacionais e não substituem avaliação profissional.

Módulo 3 — Listas, repetições e organização de dados

Listas e repetições ajudam a processar vários registros automaticamente.

Listas

pacientes = ["Luna", "Rex", "Mel", "Thor"]

print(pacientes)
print(pacientes[0])

Loop for

pacientes = ["Luna", "Rex", "Mel", "Thor"]

for paciente in pacientes:
    print(f"Gerando ficha para: {paciente}")

Loop while

tentativas = 1

while tentativas <= 3:
    print(f"Tentativa de envio do relatório: {tentativas}")
    tentativas = tentativas + 1

Módulo 4 — Funções e automação de relatórios simples

Funções permitem reaproveitar blocos de código e automatizar pequenas rotinas.

Função com parâmetros

def gerar_ficha(nome, especie, idade, peso):
    ficha = f"Paciente {nome}, espécie {especie}, {idade} anos, {peso} kg."
    return ficha

resultado = gerar_ficha("Luna", "canina", 8, 12.4)
print(resultado)

Projeto integrador: relatório automatizado

def gerar_relatorio(nome, especie, exame, achado):
    texto = f"""
Relatório simplificado

Paciente: {nome}
Espécie: {especie}
Exame: {exame}
Achado principal: {achado}

Observação: este texto é um exemplo educacional de automação.
"""
    return texto

relatorio = gerar_relatorio(
    nome="Mel",
    especie="felina",
    exame="ultrassonografia abdominal",
    achado="bexiga com conteúdo anecoico e parede preservada"
)

print(relatorio)
